SLIT:

Slit is an app that enables anyone to interact with any objects using augmented reality. The app has a number of modes which serve different purpose depending on the object. The app is based on the concept of Active vs Passive objects.

By default all objects are passive objects which can be tagged with text,images or sound. A different user running the same app can view the tagged media when the same object is detected.

An active object is any object that can communicate back with the app so that it can be remotely triggered. For example an arduino can be triggered for controlling the sensors attached to it like LEDs, speakers , servos etc.

The user can also connect multiple objects using gestures or set rules to trigger certain objects according to the state of another object. For example, two speakers are connected using hand gestures to play the same audio on both speakers. or a rule is set to dim the lights when the song is played. The possibilities are endless only limited by the active objects connected with your app.

Slit app is based on ofxQCAR which is Qualcomm's AR addon for openframeworks made possible by https://github.com/julapy/ofxQCAR

The app has the following modes:

  • Tag mode which is used to tag objects with images,sound or text.
  • Trigger mode which is used for triggering an Arduino.
  • Tuck mode which is used for playing multiplayer games present in the same physical space.
