SLIT:

Slit app is based on ofxQCAR which is Qualcomm's AR addon for openframeworks made possible by https://github.com/julapy/ofxQCAR

The app has the following modes:

Tag mode which is used to tag objects with images,sound or text Trigger mode which is used for triggering an Arduino Tuck mode which is used for playing multiplayer games present in the same physical space
